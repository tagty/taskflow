# タスク編集機能

## 概要

タスクのタイトル・タグ・期日をプロジェクト詳細画面からインラインで編集できるようにする。

## ユーザーフロー

1. タスクカードの「編集」ボタンをクリック
2. タイトル・期日・タグが編集可能な入力欄に切り替わる
3. 「保存」で更新 / 「キャンセル」で元に戻る

## 実装方針

### 変更・追加するファイル

| ファイル | 変更内容 |
|---|---|
| `src/lib/supabase/queries.ts` | `updateTask` 関数を追加 |
| `src/app/projects/[id]/actions.ts` | `updateTaskAction` を追加 |
| `src/app/projects/[id]/TaskList.tsx` | インライン編集 UI を追加 |

### データ形式

- 更新対象: `title`, `due_date`, `tags`
- `tags` はカンマ区切り文字列からパース（createTask と同じルール）

## エッジケース

- タイトルは必須（空文字は保存しない）
- 期日は任意（クリアも可能）
- タグなし（空配列）は正常ケース
