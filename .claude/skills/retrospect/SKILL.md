今日のセッションを振り返り、`.claude/learnings.md` に追記してください。

## 記録する内容

### うまくいったこと
- どんなプロンプト構造が良い出力を生んだか
- どのコンテキスト量が適切だったか（多すぎ/少なすぎ）
- 計画してから実装 vs 直接実装、どちらが効果的だったか

### うまくいかなかったこと
- 何度も修正が必要だったパターン
- コンテキストが汚染されたと感じた瞬間
- プロンプトが曖昧すぎた例

### 次回への改善
- こうすればよかった具体的な変更点

## フォーマット

```markdown
## YYYY-MM-DD

### ✓ うまくいった
- ...

### ✗ うまくいかなかった
- ...

### → 次回の改善
- ...
```

`.claude/learnings.md` が存在しない場合は新規作成してください。
