## タスク概要

`.claude/commands/evolve.md` を作成する。このコマンドは `.claude/instincts/` 内のすべての instinct ファイルを confidence 別に一覧表示し、ユーザーが手動で削除・信頼度変更・内容確認を行えるようにする。Claude Code のコマンドファイル（Markdown 形式）として実装する。

## 実装詳細

### 1. コマンドファイルの構造 🟡

**信頼性**: 🟡 *REQ-040〜041・既存コマンドファイル（diary-sync.md 等）のパターンから推測*

Claude Code のコマンドファイルは `.claude/commands/{name}.md` に配置し、`/{name}` で実行できる。

**実装ファイル**: `.claude/commands/evolve.md`

### 2. instinct 一覧表示の仕様 🔵

**信頼性**: 🔵 *REQ-040・dataflow.md フロー5より*

- `.claude/instincts/*.md` を全件読み込む
- confidence 別にグループ化して表示:
  - 🟢 high: [slug] (count: N) 最終更新: YYYY-MM-DD
  - 🟡 medium: [slug] (count: N) 最終更新: YYYY-MM-DD
  - ⚪ low: [slug] (count: N) 最終更新: YYYY-MM-DD

### 3. 管理操作の仕様 🟡

**信頼性**: 🟡 *REQ-041・ECC 設計から妥当な推測*

以下の操作をサポートする:
- **削除**: 指定した instinct ファイルを削除
- **信頼度変更**: frontmatter の confidence を手動変更
- **内容確認**: 指定した instinct ファイルの全文を表示
