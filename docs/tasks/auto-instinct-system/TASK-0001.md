## 実装詳細

### 1. 現在の settings.local.json の確認 🔵

**信頼性**: 🔵 *既存 hooks 構成より*

`.claude/settings.local.json` を読み込み、既存の hooks 設定を確認する。

**実装ファイル**: `.claude/settings.local.json`

### 2. PostToolUse hook の追加 🔵

**信頼性**: 🔵 *REQ-001・REQ-003 より*

対象ツール: `Bash`, `Edit`, `Write`, `Read`, `Glob`, `Grep` に対して `instinct-logger.sh` を PostToolUse hook として登録する。

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Bash|Edit|Write|Read|Glob|Grep",
        "hooks": [
          {
            "type": "command",
            "command": "bash .claude/hooks/instinct-logger.sh"
          }
        ]
      }
    ]
  }
}
```

### 3. Stop hook の追加 🔵

**信頼性**: 🔵 *REQ-010・architecture.md より*

既存の Stop hooks（diary-on-stop.sh、verify-on-stop.sh）に加えて `instinct-on-stop.sh` を追加する。
